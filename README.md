# MA Bioinformatics Workflows

## Install

Clone this project.  
`git clone https://...`

Setup SSH-Keypair.

Upload public SSH key to `head.hpc.zhaw.ch`.

Create an alias for task script in your `.bashrc/.zshrc`.  
`alias task='./task'`

## Usage

****
`task `

## Helpers

Convert dos file to unix file.  
`dos2unix filename`
